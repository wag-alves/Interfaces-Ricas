# 🎨 Projeto — Interfaces Ricas

Repositório destinado ao desenvolvimento da disciplina de **Interfaces Ricas**, focado na criação de interfaces modernas, interativas e usáveis.

---

## 📝 Padrão de Commits

Para manter o histórico organizado e fácil de entender, este projeto adota o padrão de **Commits Semânticos**.

### 🔥 Tipos de commit utilizados

| Tipo       | Descrição                                  |
|------------|--------------------------------------------|
| `feat`     | Nova funcionalidade                         |
| `fix`      | Correção de bugs                           |
| `docs`     | Alterações na documentação                  |
| `style`    | Formatação ou ajustes que não alteram lógica |
| `refactor` | Refatoração do código                       |
| `test`     | Adição ou correção de testes                |
| `chore`    | Tarefas gerais e manutenção do projeto     |

---

### 💡 Exemplo de mensagens

```bash
feat: adiciona tela inicial
fix: corrige erro no botão de envio
docs: atualiza README com padrão de commits
style: ajusta espaçamento no layout principal
refactor: melhora estrutura dos componentes
test: adiciona testes para componente de login
chore: atualiza dependências do projeto
