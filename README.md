# 🎨 Projeto — Interfaces Ricas

Repositório para Interfaces Ricas.

## 📝 Padrão de commits

Este projeto segue o padrão de **Commits Semânticos** para organização do histórico.

---

### 🔥 Commit semântico

- `feat`: nova funcionalidade  
- `fix`: correção de bugs  
- `docs`: alterações na documentação  
- `style`: formatação de código (sem alteração de lógica)  
- `refactor`: refatoração  
- `test`: criação ou ajuste de testes  
- `chore`: tarefas gerais e manutenção
